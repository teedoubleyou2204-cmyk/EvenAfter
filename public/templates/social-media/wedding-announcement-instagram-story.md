# Social Media Template: Wedding Announcement — Instagram Story / Reels

## Dimensions
- **Size:** 1080px × 1920px (9:16 vertical)
- **Safe Zone:** 250px from top/bottom, 80px from sides
- **Format:** Static image or video
- **Platform:** Instagram Stories, TikTok, Facebook Stories

## Style Description
Full-screen "Just Married" announcement with a romantic, cinematic feel. Features a full-bleed photo with elegant text overlay, warm gold accents, and a subtle sage gradient. Designed for maximum emotional impact.

## Design Elements

| Element            | Description                                      |
|--------------------|--------------------------------------------------|
| Full-bleed Photo   | User-supplied photo, fills entire 1080×1920      |
| Gradient Overlay   | Sage-to-transparent gradient at bottom 50%       |
| Decorative Stars   | Three small gold stars/diamonds above heading     |
| "JUST MARRIED"     | Montserrat Light, heavily tracked (8px letter-spacing) |
| Couple Names       | Playfair Display Bold, white, drop shadow         |
| Gold Heart Divider | Gold heart or diamond symbol between sections     |
| Quote              | Cormorant Garamond Italic, warm gold              |
| CTA Footer         | Semi-transparent sage bar, rounded corners         |

## Color Application
- **Background:** Full-bleed photo with overlay
- **Primary text:** White (#FFFFFF) for readability on photo
- **Accent text:** Warm Gold (#C9A94E)
- **Quote:** Warm Gold (#C9A94E)
- **CTA bar fill:** Sage (#9CAF88) at 85% opacity
- **Gradient overlay:** Sage (#9CAF88) at 60% fading to 0%

## Customization Notes
- The photo should be high-contrast or well-lit — text readability on the overlay is critical
- For video/Reels: animate the decorative stars twinkling in, then the names fading in
- Add a subtle "typewriter" effect for the quote appearing letter by letter
- Alternative: use a video clip (first kiss, walk down the aisle) as the background

## Production Specs
- **File format:** PNG (static), MP4 (video — 1080×1920, 10-20 sec)
- **Resolution:** 1080×1920px @ 72 DPI
- **Color mode:** RGB
- **Text safety:** Avoid top 250px (iOS status bar) and bottom 200px (swipe-up area)
