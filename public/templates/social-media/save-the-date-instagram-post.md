# Social Media Template: Save the Date — Instagram Post (Square)

## Dimensions
- **Size:** 1080px × 1080px (1:1 square)
- **Safe Zone:** 100px from all edges
- **Format:** Static image or short video cover
- **Platform:** Instagram feed, Facebook feed

## Style Description
Elegant square announcement card matching the Classic Invitation style. Features a sage green border, centered typography, and a subtle gold decorative element. Optimized for mobile viewing with large, readable text.

## Design Elements

| Element            | Description                                      |
|--------------------|--------------------------------------------------|
| Outer Border       | Sage (#9CAF88) solid border, 15px thick, 40px from edge |
| Inner Frame        | Thin gold (#C9A94E) rule, 1px, 30px from sage border |
| Header Text        | "SAVE THE DATE" — Montserrat Light, uppercase, tracked |
| Couple Names       | Playfair Display Bold — largest text on canvas   |
| Date               | Numeric format (DD · MM · YYYY) in Sage         |
| Gold Divider       | Horizontal rule with diamond center, Warm Gold  |
| Footer             | Montserrat Light, smaller size, #handle + hashtag |
| Background         | Ivory (#FDF8F0) solid fill or subtle texture     |

## Color Application
- **Background:** Ivory (#FDF8F0)
- **Outer border:** Sage (#9CAF88)
- **Inner rule & header text:** Warm Gold (#C9A94E)
- **Couple names & venue:** Charcoal (#3A3A3A)
- **Date:** Sage (#9CAF88)
- **Handle/hashtag:** Sage (#9CAF88)

## Customization Notes
- Replace the couple names, date, venue, and handle with your own
- For a more personal touch, replace the ivory background with an engagement photo (use photo as background with white text overlay)
- Swap gold accents to rose gold (#E8B4B8) or silver (#A8A8A8)
- Add the venue location as an animated map pin for video/Reels versions

## Production Specs
- **File format:** PNG (static), MP4 (Reels — 1080×1080, 5-15 sec)
- **Resolution:** 1080×1080px @ 72 DPI (web-optimized)
- **Color mode:** RGB
- **Max file size:** < 5MB for optimal Instagram upload
- **Text minimum:** 14px (ensure readability on mobile)
