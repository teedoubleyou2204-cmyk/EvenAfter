# Social Media Template: Wedding Announcement — Instagram Post (Square)

## Dimensions
- **Size:** 1080px × 1080px (1:1 square)
- **Safe Zone:** 100px from all edges
- **Format:** Static image
- **Platform:** Instagram feed, Facebook feed

## Style Description
Celebratory announcement post to share the big news after the wedding (or before, as an engagement announcement). Features a modern asymmetric split with a photo on one side and the announcement details on the other.

## Design Elements

| Element            | Description                                      |
|--------------------|--------------------------------------------------|
| Left Panel         | Photo area — 40% width, full height              |
| Left Overlay       | Semi-transparent sage gradient over photo       |
| Gold Icon          | Diamond or ring icon overlay on photo, gold      |
| Right Panel        | White (#FFFFFF) background — 60% width           |
| Vertical Divider   | 1px gold line between panels                     |
| "MRS & MR"         | Montserrat Light, small, all caps, tracked       |
| Last Names         | Playfair Display Bold, largest text               |
| Month/Year         | Montserrat Bold, uppercase, sage color            |
| Hashtag            | Montserrat Light, warm gold                       |

## Color Application
- **Left panel:** Photo (user-supplied) with sage gradient overlay
- **Right panel background:** White (#FFFFFF)
- **"MRS & MR":** Warm Gold (#C9A94E)
- **Names:** Charcoal (#3A3A3A)
- **Date:** Sage (#9CAF88)
- **Hashtag:** Warm Gold (#C9A94E)

## Customization Notes
- Swap the photo for a wedding party group shot, a detail shot (rings, bouquet), or the venue
- For engagement announcements: change "MRS & MR" to "FUTURE MRS & MR" or "SAYING YES"
- The gold icon on the photo can be a heart, ring, or venue-appropriate symbol
- Alternative layout: swap panels (text left, photo right) for variety in a series
