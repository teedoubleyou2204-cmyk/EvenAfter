# Schedule Template: Wedding Day Timeline

## Dimensions
- **Print Size:** 8.5″ × 11″ (US Letter / A4 equivalent)
- **Bleed:** 0.125″ (3mm) on all sides
- **Safe Zone:** 0.5″ (12mm) from edge
- **Layout:** Portrait (vertical) — single-sided

## Style Description
Clean, scannable timeline layout with a subtle decorative header. Each time block is clearly separated with a soft sage accent bar. Perfect for printing and handing to bridal party, vendors, and family members.

## Design Elements

| Element         | Description                                         |
|-----------------|-----------------------------------------------------|
| Header Banner   | Sage (#9CAF88) thin rule top/bottom, centered text  |
| Time Blocks     | White card with soft sage left accent bar (3pt)     |
| Time Icons      | Clock emoji or small circle icon next to each time  |
| Alternating Rows| Every other block has light blush horizontal stripe |
| Footer Bar      | Double thin rule (sage/gold) with coordinator info  |
| Background      | White (#FFFFFF)                                     |

## Typography

| Text Level         | Font              | Size   | Color       | Alignment |

| Footer             | Montserrat        | 8pt    | Sage        | Center    |

## Customization Notes
- Add or remove time blocks as needed (drag-and-drop in Canva)
- Swap the sage accent bar color to match your wedding palette
- The left margin "⏰" can be replaced with a custom icon or small photo
- Print one-sided and trim to fit in a 9×12 envelope for the wedding party

- **File format:** PDF (print-ready) / PNG
