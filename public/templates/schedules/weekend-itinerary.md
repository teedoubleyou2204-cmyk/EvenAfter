# Schedule Template: Weekend Itinerary

## Dimensions
- **Print Size:** 8.5″ × 11″ (US Letter) — folds to 3.67″ × 8.5″ tri-fold brochure
- **Bleed:** 0.125″ (3mm) on all sides
- **Safe Zone:** 0.35″ (9mm) from edge
- **Layout:** Landscape, tri-fold (3 panels)

## Style Description
Bi-fold or tri-fold brochure layout covering an entire wedding weekend (rehearsal, ceremony, brunch, etc.). Each day gets a panel with a sage header block. Perfect for destination weddings or multi-day celebrations.

## Design Elements

| Element         | Description                                         |
|-----------------|-----------------------------------------------------|
| Cover Panel     | Sage block with white text, centered layout         |
| Day Headers     | Montserrat uppercase, warm gold, thin rule above    |
| Event Items     | Time bolded left, description next, location below  |
| Panel Dividers  | 0.5pt sage vertical rules between panels            |
| Background      | White (#FFFFFF)                                     |

## Customization Notes
- For 2-day weddings, remove the third panel or use it for maps/directions
- Add a QR code on the cover panel linking to your wedding website
- Include a small map on the back panel for venue-to-venue directions
- Print on heavyweight paper (80lb+), fold carefully with a bone folder for crisp creases

## Production Specs
- **File format:** PDF (print-ready, with fold marks)
- **Resolution:** 300 DPI
- **Paper:** 80lb (170gsm) matte cover stock
- **Finish:** Tri-fold, fits standard #10 envelope or pocket
