# Reception Pack: Table Number

## Dimensions
- **Print Size:** 5″ × 7″ (127mm × 178mm) — standard tent easel
- **Alternative:** 4″ × 6″ for smaller tables
- **Bleed:** 0.125″ (3mm) on all sides
- **Safe Zone:** 0.25″ (6mm) from edge
- **Layout:** Portrait (vertical) — double-sided

## Style Description
Elegant table number card designed to sit in a small easel or frame. Features a large number in Playfair Display, the couple's names at the top, and an optional back side with the menu or a thank-you note.

## Customization Notes
- For the back side: print a thank-you note, a fun fact about the table, or the menu
- For table names instead of numbers, simply replace the numeral with text
- Print as double-sided or single-sided in an easel frame
