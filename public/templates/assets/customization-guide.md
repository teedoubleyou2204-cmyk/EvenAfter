### How to Swap Colors

1. **In Canva:** Select any element → Click the color swatch → Enter your hex code → Apply globally using "Change all" where available.
2. **In Adobe InDesign:** Use the Swatches panel → Double-click → Enter new CMYK or hex values → Replace all instances.

## 3. Typography Guide

### Recommended Fonts (Google Fonts — Free & Canva-Compatible)

| Mood            | Heading Font                     | Body Font                       |
|-----------------|----------------------------------|---------------------------------|
| Classic         | Playfair Display                 | Cormorant Garamond              |
| Modern          | Montserrat Light                 | Lora                            |
| Minimalist      | EB Garamond                      | Work Sans                       |
| Rustic          | DM Serif Display                 | Josefin Sans                    |
| Playful         | Pacifico                         | Nunito Sans                     |
| Romantic        | Great Vibes                      | Libre Baskerville               |

### Font Pairing Rules
- **One display font + one body font:** Don't use more than two font families
- **Contrast is key:** Pair a serif heading with a sans-serif body (or vice versa)
- **Size hierarchy:** Headings 3–4× larger than body text for clear visual structure
- **Italics sparingly:** Use for secondary info only (venue, "together with", dates)

### Print Settings
- **Resolution:** 300 DPI minimum
- **Color Mode:** CMYK for professional printing; RGB for at-home or digital use
- **Paper Weight Guide:**
  - Invitations: 110lb–130lb (300–350gsm) — matte, linen, or felt finish
  - Schedules/Checklists: 32lb–60lb (120–170gsm) — premium laser or matte cover
  - Seating Chart: 100lb+ (270gsm+) cover stock or mount on foam core
  - Escort Cards: 110lb (300gsm) matching invitation paper

## 5. Personalization Steps (Canva Users)

### Step-by-Step Process

1. **Open in Canva**
   - Upload the PDF template to Canva or use the matching Canva template link
   - Canva automatically detects the correct dimensions

2. **Replace All Placeholder Text**
   - Click any text box → Select all (Ctrl+A/Cmd+A) → Type your text
   - Update: names, dates, venue, RSVP info, schedule times

3. **Swap Fonts (Optional)**
   - Highlight text → Click font dropdown → Search our recommended fonts
   - Stick to the heading/body pairing in the style guide

4. **Update Colors**
   - Click colored element → Click color swatch → Enter your hex code
   - Use the "Change all" option for consistency

5. **Add Photos**
   - Templates with photo placeholders: Click the image area → Upload your photo
   - Engagement photos, venue shots, or couple portraits work beautifully

6. **Review & Export**
   - Check all spelling and dates
   - Export as **PDF Print** (for professional printing) at 300 DPI
   - Export as **PNG** (for digital sharing, social media, email)

## 6. Creating a Cohesive Suite

### The "Golden Thread" Principle
Choose **one or two consistent elements** that appear across every piece:

| Element        | Examples                                        |
|----------------|-------------------------------------------------|
| A motif        | Floral corner, monogram, gold diamond           |
| A color        | Sage header on every page                       |
| A font pairing | Playfair + Montserrat across all pieces         |
| A border style | Thin gold frame on invitations + escort cards   |

### Recommended Suite Composition
```
□ Invitation Card (5×7)
□ RSVP Card (3.5×5) — or digital RSVP via website
□ Details Card (3.5×5) — accommodations, registry
□ Wedding Day Schedule (8.5×11) — for wedding party
□ Weekend Itinerary (folds to 3.67×8.5) — for guests
□ Seating Chart Display (24×36 poster)
□ Escort Cards (one per guest, 3.5×5)
□ Planning Checklist (for you!)
□ Vendor Checklist (for coordination)
```

## 7. Digital vs. Print Checklist

### For Professional Printing
- [ ] Convert all colors to CMYK
- [ ] Add 0.125″ bleed on all sides
- [ ] Set all text to at least 8pt (smaller may not print clearly)
- [ ] Embed all fonts or convert text to outlines
- [ ] Export as PDF/X-1a or "PDF Print — High Quality"
- [ ] Choose paper stock (request a sample pack from your printer)
- [ ] Order test print before bulk order

### For Home Printing
- [ ] Use 300 DPI or higher printer setting
- [ ] Select "Borderless Printing" if available
- [ ] Use premium inkjet or laser paper (not standard copy paper)
- [ ] Trim excess with a paper cutter for clean edges
- [ ] Test print on plain paper first to check alignment

### For Digital Sharing
- [ ] Convert to RGB color mode
- [ ] Export as PNG or JPG (not PDF)
- [ ] Optimize file size (< 5MB for email)
- [ ] Consider a watermark-free version for your wedding website

## 8. Troubleshooting

| Problem                          | Solution                                           |
|----------------------------------|----------------------------------------------------|
| Text looks too small when printed | Check your printer's scaling setting — ensure "Actual Size" (not "Fit to Page") |
| Colors look different on screen  | Screens are RGB, print is CMYK — accept slight variation; request a proof from your printer |
| Canva template doesn't match PDF | Download the PDF from our store and upload to Canva directly |
| Bleed marks visible              | Trim to the final size using crop marks (included in file) |
| Font not available in Canva      | Use the closest match from our recommended alternatives table |
