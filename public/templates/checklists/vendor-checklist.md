# Checklist Template: Wedding Vendor Coordination

## Dimensions
- **Print Size:** 8.5″ × 11″ (US Letter)
- **Bleed:** none needed (borderless OK)
- **Safe Zone:** 0.5″ (12mm) from edge
- **Layout:** Portrait (vertical) — grid/table format

## Style Description
At-a-glance vendor contact sheet combined with confirmation checklist. Each vendor gets a row block with contact info, booking status, payment schedule, and key dates. Designed to be printed and kept in a wedding planner binder.

## Design Elements

| Element           | Description                                       |
|-------------------|---------------------------------------------------|
| Header            | Double rule with centered title in Playfair       |
| Table Grid        | Thin lines separating rows and columns (0.5pt)    |
| Header Row        | Sage (#9CAF88) fill, white text, bold             |
| Checkbox Column   | First column — circle checkbox (☐)               |
| Payment Sub-rows  | Indented lines for deposit vs. balance            |
| Footer            | Budget summary with dollar amount placeholders    |
| Background        | White (#FFFFFF)                                   |

## Typography

## Customization Notes
- Add rows for additional vendors (videographer, baker, planner, etc.)
- Convert to a shared digital doc (Google Sheets) for real-time collaboration
- Add a "Due Date" column for payment deadlines
- Include a "Confirmed?" checkbox column for booking status

## Production Specs
- **File format:** PDF (print-ready) / spreadsheet-compatible CSV version
- **Resolution:** 300 DPI
- **Paper:** 32lb (120gsm) premium laser paper
- **Use:** Binder insert or clipboard reference sheet
