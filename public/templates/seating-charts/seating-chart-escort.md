# Seating Chart Template: Escort Cards Layout

## Dimensions
- **Print Size:** 5″ × 7″ (127mm × 178mm) — single escort card
- **Bleed:** 0.125″ (3mm) on all sides
- **Safe Zone:** 0.25″ (6mm) from edge
- **Layout:** Portrait (vertical)

## Style Description
Individual escort card design with table assignment. Each guest receives a card with their name and table number. Designed as a template that can be duplicated and personalized for each guest. Works for 30–200+ guests.

## Design Elements

| Element              | Description                                    |
|----------------------|------------------------------------------------|
| Top Motif            | Small circular frame (1.2″) with gold border   |
| Gold Border          | Ornate thin frame around card, 0.5pt, gold     |
| Guest Name           | Large serif text, centered                     |
| "TABLE" + Number     | Montserrat Bold for "TABLE", large number      |
| Meal Selection Box   | Small rounded rectangle, thin sage stroke      |
| Background           | Ivory (#FDF8F0)                                |

## Production Specs
- **File format:** PDF (print-ready, merge-ready for mail merge)
- **Card size:** 5″ × 7″ — can be trimmed to 3.5″ × 5″ for smaller cards
- **Paper:** 110lb (300gsm) matte or felt cardstock
- **Display suggestion:** Arrange alphabetically on a table near the reception entrance, or clip to a hanging board
