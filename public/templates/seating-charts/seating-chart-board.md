# Seating Chart Template: Large Display Board

## Dimensions
- **Print Size:** 24″ × 36″ (609mm × 914mm) — large poster
- **Alternative Sizes:** 18″ × 24″ or 36″ × 48″ (scale proportionally)
- **Safe Zone:** 0.75″ (19mm) from edge
- **Layout:** Portrait (vertical) — grid layout

## Style Description
Large-format display chart showing all tables and assigned guests at a glance. Organized as a grid of table "nodes" with guest names listed below each table number. Perfect for printing and mounting on foam core or framing in a large easel.

## Design Elements

| Element           | Description                                       |
|-------------------|---------------------------------------------------|
| Header            | "Welcome to Our Wedding" — Playfair Display 30pt  |
| Subheader         | "Find Your Seat" — Montserrat Light 14pt          |
| Table Boxes       | Rectangular boxes with thin sage border (1pt)     |
| Table Header      | Sage (#9CAF88) fill on table number header        |
| Guest Names       | Montserrat Regular 10pt, centered, single column  |
| Sweetheart Table  | Larger box, gold border (1.5pt), centered layout  |
| Footer            | Double gold-and-sage rules with thank-you message |

## Color Application
- **Table box border:** Sage (#9CAF88)
- **Table header fill:** Sage (#9CAF88), text: White
- **Sweetheart table border:** Warm Gold (#C9A94E)
- **Guest names:** Charcoal (#3A3A3A)
- **Background:** Ivory (#FDF8F0)

## Customization Notes
- For large weddings (200+ guests), scale to 36×48 and increase font size proportionally
- The sweetheart table is centered and visually elevated — swap for a head table layout if needed
- Table boxes can be rearranged to match the actual floor plan layout (round tables vs. long tables)
- Add table icons or room markers for multi-room venues

## Production Specs
- **File format:** PDF (print-ready, tiled for home printers or single file for print shop)
- **Resolution:** 300 DPI
- **Color mode:** CMYK
- **Printing:** Print shop (Staples, FedEx Office, or local printer)
- **Mounting:** Foam core board (3/16″ or 1/2″) or acrylic frame
- **Display:** Easel stand, wall mount, or window frame
