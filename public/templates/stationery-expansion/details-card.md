# Stationery Expansion: Details / Information Card

## Dimensions
- **Print Size:** 3.5″ × 5″ (89mm × 127mm) — matching RSVP
- **Bleed:** 0.125″ (3mm) on all sides
- **Safe Zone:** 0.25″ (6mm) from edge
- **Layout:** Portrait (vertical)

## Style Description
Compact information card with wedding weekend details, accommodation info, and registry links. Uses a clean two-column layout for maximum information in a small space.

## Customization Notes
- Include a QR code to your wedding website for easy access
- Add a small map on the back side showing venue locations
- Remove sections that don't apply (e.g., no hotel block)
- For destination weddings, add a "Things to Do" section
