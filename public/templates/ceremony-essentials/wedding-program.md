# Ceremony Essentials: Wedding Program / Booklet

## Dimensions
- **Print Size:** 5.5″ × 8.5″ (140mm × 216mm) — half-letter, folded
- **Alternative:** 4″ × 9″ for fan programs
- **Bleed:** 0.125″ (3mm) on all sides
- **Layout:** Landscape — bi-fold (4 panels: cover, inside left, inside right, back)

## Style Description
Elegant ceremony program booklet with a sage cover featuring the couple's names and date. Inside contains the ceremony order, readings, music selections, and acknowledgments. Designed as a bi-fold for easy printing at home.

## Design Elements

| Element           | Description                                        |
|-------------------|----------------------------------------------------|
| Cover Background  | Sage (#9CAF88) solid fill                          |
| Cover Text        | White (#FFFFFF) — names, date, venue               |
| Cover Motif       | Gold floral icon at top of cover                   |
| Section Headers   | Montserrat Bold, uppercase, warm gold              |
| Section Rules     | Thin gold rules                                    |
| Body Text         | Cormorant Garamond, charcoal, left-aligned         |

| Back Cover        | Ivory (#FDF8F0) with couple names                  |

## Customization Notes
- For outdoor/casual weddings, reduce to a single-page (front/back) program
- Add a fan handle at the bottom for warm-weather ceremonies (cutout shape)
- Include a QR code linking to your wedding website or reception details
- The wedding party section can be moved to the back if needed

## Production Specs
- **File format:** PDF (print-ready, booklet imposition)
- **Resolution:** 300 DPI
- **Paper:** 80lb (170gsm) matte cover stock
- **Size:** 5.5″ × 8.5″ (folded from 8.5″ × 11″)
- **Binding:** Saddle-stitch or bi-fold
