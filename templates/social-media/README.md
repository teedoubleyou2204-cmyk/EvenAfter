## Posting Best Practices

### Optimal Posting Times (Wedding Niche)
- **Instagram Feed:** Tuesday–Thursday, 10am–2pm
- **Instagram Stories:** Evenings, 7pm–9pm
- **Facebook:** Wednesday–Friday, 9am–1pm
- **TikTok:** Tuesday–Thursday, 7am–10am

## Customization Summary

All templates share common customization rules:
1. Replace `[Couple Names]`, `[Date]`, `[Venue]`, `[Handle]`
2. Swap colors via hex values (see color palette above)
3. Add photos to designated photo areas
4. Export at 72 DPI, RGB, PNG for web/social
