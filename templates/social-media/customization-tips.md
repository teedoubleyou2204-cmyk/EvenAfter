- **Keep the focal point** in the center or right third (avoid left edge on Facebook banners — profile pic covers it)

## 2. Color Swapping Guide

### How to Swap Colors in Canva
1. Select the element you want to recolor
2. Click the color swatch in the top toolbar
3. Enter your hex code (e.g., `#9CAF88` for Sage)
4. Use "Change all" for bulk updates

## 3. Typography on Social Media

### Readability Rules
- **Headlines:** Minimum 28px on Stories (larger is better)
- **Body text:** Minimum 16px on Stories, 14px on feed posts
- **Contrast ratio:** Ensure text has a strong contrast against background
- **Text overlay on photos:** Always add a semi-transparent overlay or drop shadow

### Font Access
All templates use **Google Fonts** (Playfair Display, Cormorant Garamond, Montserrat) which are available in:
- **Canva:** Search by name in the text tool
- **Adobe Express:** Available in the fonts panel
- **Photoshop / Illustrator:** Free download from Google Fonts
- **CapCut (for video):** Download and install

## 4. Platform-Specific Best Practices

### Instagram
- **Feed Posts:** Use 1080×1080 (square) for best grid appearance
- **Stories:** Keep text within 80-1000px horizontal, avoid top 250px (system UI)
- **Reels:** Use 1080×1920, keep text in center 60% of frame
- **Carousels:** 2-10 slides; use the countdown series as a 3-slide carousel
- **Alt Text:** Add descriptive alt text to every post for accessibility

### Facebook
- **Cover Photo:** Leave bottom-left 170×170px clear for profile picture
- **Feed Posts:** 1200×630px recommended; our 1080×1080 works well too
- **Stories:** Same specs as Instagram Stories (1080×1920)

### TikTok
- **Videos:** 1080×1920, 15-60 seconds, vertical
- **Text:** Larger than Instagram (mobile-first, younger audience)
- **Trends:** Adapt the countdown series as a "POV" trend video

### Pinterest
- **Pins:** 1000×1500px (2:3 ratio) recommended for best visibility
- **Description:** 150-300 characters with keywords
- **Our templates:** The 1080×1080 posts work as square pins

## 7. Accessibility Tips
- All text should have **WCAG AA** contrast ratio (4.5:1 minimum)
- Add **alt text** describing the image and the text content
- Use **sentence case** for body text (easier to read than all caps)
- Avoid **red/green** color differentiation (colorblind users)
- Include **closed captions** on all video content
