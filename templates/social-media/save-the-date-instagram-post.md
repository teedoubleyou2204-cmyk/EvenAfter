# Social Media Template: Save the Date — Instagram Post (Square)

## Dimensions
- **Size:** 1080px × 1080px (1:1 square)
- **Safe Zone:** 100px from all edges
- **Format:** Static image or short video cover
- **Platform:** Instagram feed, Facebook feed

## Style Description
Elegant square announcement card matching the Classic Invitation style. Features a sage green border, centered typography, and a subtle gold decorative element. Optimized for mobile viewing with large, readable text.

## 🎨 Color Customization (How to Change Everything)

**This design is fully customizable — no color is locked.** Every element can be swapped to match your wedding palette.

### Color Swap Mapping

| If You Want To Change | Default     | Swap To (Example)               | How (Canva)           |
|-----------------------|-------------|----------------------------------|----------------------|
| Background fill       | #FDF8F0     | Photo, #FFFFFF, #F5E6D3         | Click bg → Color     |
| Outer border          | #9CAF88     | #2C3E50 Navy, #E8C4C4 Blush     | Click border → Color |
| Header accent text    | #C9A94E     | #E8B4B8 Rose, #C0C0C0 Silver    | Select text → Color  |
| Date color            | #9CAF88     | Match to your primary or accent | Select date → Color  |

## Customization Notes
- Replace the couple names, date, venue, and handle with your own
- The entire color scheme is swappable — change one element or all of them
- For photo backgrounds, ensure high contrast (dark overlay + white text, or light overlay + dark text)

## Production Specs
- **File format:** PNG (static), MP4 (Reels — 1080×1080, 5-15 sec)
- **Resolution:** 1080×1080px @ 72 DPI (web-optimized)
- **Color mode:** RGB
- **Max file size:** < 5MB for optimal Instagram upload
- **Text minimum:** 14px (ensure readability on mobile)
