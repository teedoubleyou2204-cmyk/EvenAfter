# Social Media Template: Save the Date — Instagram Story / Reels

## Dimensions
- **Size:** 1080px × 1920px (9:16 vertical)
- **Safe Zone:** 250px from top/bottom, 80px from sides
- **Top Text Safe Zone:** Below 250px from top (avoid system UI overlap)
- **Format:** Static image or short video
- **Platform:** Instagram Stories, TikTok, Facebook Stories, Snapchat

## Style Description
Full-screen vertical announcement with a split layout. The top 40% features a photo or solid sage color block with the couple's names, and the bottom 60% has the save-the-date details. Designed for quick consumption and easy sharing.

## Design Elements

| Element            | Description                                      |
|--------------------|--------------------------------------------------|
| Top Section        | Full-width photo or sage/blush gradient (40%)    |
| Photo Overlay      | Semi-transparent black or ivory gradient overlay |
| Names on Photo     | Playfair Display, white or ivory, drop shadow    |
| Bottom Section     | Ivory (#FDF8F0) background (60%)                 |
| Gold Divider       | Thin gold rule, 60px from left, 40% width        |
| CTA Button Area    | Rounded rectangle, sage fill, white text          |
| Swipe-up Text      | "Link in bio" — standard Stories CTA             |

## 🎨 Color Customization

**This template is color-flexible — every element can be swapped.** The two main color zones are the bottom section (Ivory background) and the accent elements (Gold, Sage).

### Color Swap Mapping

| Design Element             | Default     | Swap To (Example)                     |
|----------------------------|-------------|----------------------------------------|
| Bottom background          | #FDF8F0     | White, cream, or transparent (photo)  |
| "SAVE THE DATE" text       | #C9A94E     | White, Sage, Navy, Blush              |
| CTA button fill            | #9CAF88     | Navy, Terracotta, Blush, Gold         |
| Date / venue text          | #3A3A3A     | Keep dark for readability             |
| Handle                     | #9CAF88     | White (if on photo), or your accent   |

### Quick Step
1. **CTA button:** Click button → Color → Your wedding primary
2. **Accent text:** Select "SAVE THE DATE" → Color → Your accent
3. **Bottom section:** Swap ivory for a matching tone or extend the photo full-bleed

## Customization Notes
- The top photo is the hero — use an engagement photo, a shot of the venue, or a flat-lay of rings/invitations
- For video/Reels: animate the date "flipping in" or the divider drawing across
- Add a gentle parallax effect for Reels (photo moves slightly while text stays still)
- Alternative: use a sage-to-blush gradient instead of a photo for a more graphic look

## Production Specs
- **File format:** PNG (static), MP4 (Reels/Stories — 1080×1920, 15 sec max)
- **Resolution:** 1080×1920px @ 72 DPI
- **Color mode:** RGB
- **Safe zones:** Keep all text within 80–1000px from sides, avoid top 250px (system UI) and bottom 200px (swipe-up/slider area)
- **Video length:** 5–15 seconds for Reels, up to 60 seconds for Stories
