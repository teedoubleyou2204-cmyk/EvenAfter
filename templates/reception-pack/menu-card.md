# Reception Pack: Menu Card

## Dimensions
- **Print Size:** 4″ × 6″ (101mm × 152mm) — standard menu
- **Alternative:** 5″ × 7″ for larger menus
- **Bleed:** 0.125″ (3mm) on all sides
- **Safe Zone:** 0.25″ (6mm) from edge
- **Layout:** Portrait (vertical)

## Style Description
Elegant menu card with a sage border frame, gold divider accents, and clean serif typography. Features a 3-course layout (starter, main, dessert) with space for wine pairings and a personalized message from the couple.

## 🎨 Color Customization

**Every color is swappable.** The border, course headers, and sage text are all customizable.

| Design Element     | Default     | Swap To (Example)                |
|--------------------|-------------|----------------------------------|
| Card background    | #FDF8F0     | White, Cream, or photo           |
| Outer border       | #9CAF88     | Navy, Terracotta, Gold           |
| Course headers     | #C9A94E     | Sage, Blush, Silver              |
| Course divider     | #C9A94E     | Rose Gold, or match border       |
| Date text          | #9CAF88     | Your accent color                |

## Customization Notes
- Add or remove courses as needed (2-course or 4-course options)
- Replace wine pairing with cocktail or non-alcoholic beverage options
- For plated dinners, add columns for "Meat / Fish / Vegetarian" choices
- The outer gold border can be removed for a cleaner, borderless look
