# Checklist Template: Weekend Packing List

## Dimensions
- **Print Size:** 8.5″ × 11″ (US Letter)
- **Safe Zone:** 0.5″ (12mm) from edge
- **Layout:** Portrait (vertical) — 2-column checklist

## Style Description
Split-column packing list with categories for bride, groom, and miscellaneous essentials. Each category has a colored header bar and checkable items. Compact two-column layout to fit everything on one page.

## 🎨 Color Customization

This two-column packing checklist uses Sage header, Gold accents, and Charcoal dividers — all customizable.

### Color Swap Mapping

| Design Element             | Default     | Swap To (Example)                |
|----------------------------|-------------|-----------------------------------|
| Header block fill           | #9CAF88 Sage | Navy, Blush, Terracotta          |
| Header block text           | #FFFFFF White | Keep white (on dark fill)       |
| Bride column header rule   | #C9A94E Gold  | Accent color or primary          |
| Groom column header rule   | #3A3A3A Charcoal | Match to bride column        |
| Essentials section border  | #9CAF88 Sage  | Primary or accent                |
| Checkbox stroke            | #9CAF88 Sage  | Primary or accent                |
| Tip footer                  | #C9A94E Gold  | Warm tone or your accent         |

### Quick Step
1. Click the sage header block → Color → Your wedding primary
2. Click the gold rule under "BRIDE" → Color → Your accent
3. Click one checkbox circle → Select all same → Change all at once

## Customization Notes
- Edit items to match your specific wedding weekend needs
- Add a third column for "Pets/Kids" if needed
- The emoji column headers can be swapped for small icons or initials
