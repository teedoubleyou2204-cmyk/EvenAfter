# Invitation Template: Minimalist Line Art

## Dimensions
- **Print Size:** 5″ × 7″ (127mm × 178mm) — standard invitation card
- **Bleed:** 0.125″ (3mm) on all sides
- **Safe Zone:** 0.25″ (6mm) from edge
- **Layout:** Portrait (vertical) — centered

## Style Description
Ultra-minimal design featuring hand-drawn line art (floral border or couple silhouette), generous whitespace, and restrained typography. No color fills — relies on line weight and hierarchy. Ideal for elopements, micro-weddings, and design-conscious couples.

## Design Elements

| Element        | Description                                         |
|----------------|-----------------------------------------------------|
| Line Art Frame | Hand-drawn style floral corner ornaments in charcoal|
| Center Text    | Clean hierarchy with generous line spacing (2x)     |
| Date           | Numeric format with dots (DD · MM · YYYY)          |
| RSVP Box       | Thin charcoal stroke, rounded corners (4pt radius) |
| Whitespace     | Minimum 0.75″ margins on all sides                 |
| Paper Color    | White or cream paper acts as the only "color"      |

### Signature Accent Option
Add a single accent color to just one element (e.g., the date line) for a subtle pop:
- Sage (#9CAF88), Terracotta (#C87D6E), Navy (#2C3E50), Blush (#E8C4C4), Gold (#C9A94E)

## 🎨 Color Customization (How to Change Everything)

**This design starts monochrome but can be colored in countless ways.** Because it's minimal, even one color change creates a dramatic shift.

### Ways to Add Color

| Approach              | What to Change                                     | Effect                        |
|-----------------------|----------------------------------------------------|-------------------------------|
| 🌿 Single accent      | Change date line text color only                   | Subtle pop, keeps minimalism  |
| 🎨 Two-color          | Add accent to date + change floral line art        | Modern graphic feel           |
| 🖌 Full recolor       | Replace all monochrome (Charcoal → Navy, Gold)     | Transforms to full-color look |
| 📜 Foil stamping      | Use copper/gold/rose gold foil instead of ink       | Luxury letterpress finish     |

## Customization Notes
- The floral corner art can be replaced with custom line art (venue sketch, couple portrait outline, botanical elements)
- Great for DIY letterpress or foil stamping — the minimal line art translates perfectly
- For a warmer feel, use cream/ivory paper stock
- Add a subtle envelope liner (blush or sage pattern) for impact without cluttering the card itself

## Production Specs
- **File format:** PDF (print-ready) / PNG (digital) / SVG (for letterpress)
- **Resolution:** 300 DPI
- **Color mode:** Monochrome (1-color) + optional spot color
- **Paper:** 110lb (300gsm) white or cream textured cardstock
- **Specialty print:** Letterpress, foil stamping (gold or copper), or thermography
- **Envelope:** A7 (5.25″ × 7.25″) — white with blush liner
