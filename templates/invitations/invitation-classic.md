# Invitation Template: Classic Elegance

## Dimensions
- **Print Size:** 5″ × 7″ (127mm × 178mm) — standard invitation card
- **Bleed:** 0.125″ (3mm) on all sides
- **Safe Zone:** 0.25″ (6mm) from edge
- **Layout:** Portrait (vertical)

## Style Description
Traditional symmetrical layout with a central focus. Features an ornate gold-toned border frame, elegant script heading, and clean serif body text. Ideal for formal evening weddings.

## 🎨 Color Customization (How to Change Everything)

**This design supports full color customization — no color is locked.** Every hex value above can be swapped to match your wedding palette.

### Step-by-Step (Canva)
1. **Card background:** Select the card area → Click color swatch → Enter your paper/card color hex
2. **Text colors:** Click any text element → Click text color → Enter new hex (use "Select all text of same color" for bulk changes)
3. **Borders & dividers:** Click the gold line/border → Click color swatch → Enter new accent color
4. **Floral motif:** Click the motif element → Click color → Match to your accent color

## Customization Notes
- Replace `[NAME OF COUPLE]` with the couple's full names (e.g., "Emily Rose & James Alexander")
- Update date, time, venue, and RSVP info
- **The entire color scheme is designed to be swappable.** Change one color or all five — the design works beautifully with any combination.
- The floral motif at top can be replaced with a custom monogram or left as a decorative frame

## Production Specs
- **File format:** PDF (print-ready) / PNG (digital)
- **Resolution:** 300 DPI
- **Color mode:** CMYK for print, RGB for digital
- **Paper:** 110lb (300gsm) matte cardstock
- **Envelope:** A7 (5.25″ × 7.25″) — matching ivory or sage
