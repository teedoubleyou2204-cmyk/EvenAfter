# Stationery Expansion: RSVP Card

## Dimensions
- **Print Size:** 3.5″ × 5″ (89mm × 127mm) — standard reply card
- **Bleed:** 0.125″ (3mm) on all sides
- **Safe Zone:** 0.25″ (6mm) from edge
- **Layout:** Portrait (vertical)

## Style Description
Elegant RSVP card with a check-box format for meal selection, attendance, and a line for guest names. Matches the Classic Invitation style with sage border and gold accent.


## Customization Notes
- Add space for "Song Requests" or "Dietary Restrictions" for a personal touch
- For digital RSVPs, omit this card and include a QR code on the invitation instead
- The checkbox format can be replaced with lines for written responses

## Production Specs
- **File format:** PDF (print-ready)
- **Resolution:** 300 DPI
- **Paper:** 80lb (170gsm) matte cardstock
- **Envelope:** A2 (4.375″ × 5.75″) — matching invitation suite
